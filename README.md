## Чем VelumVPN отличается от Koala Clash

VelumVPN — это форк [koala-clash](https://github.com/coolcoala/koala-clash) с расширенными возможностями, ориентированными на русскоязычных пользователей и простоту настройки.

### Диагностика соединений
Встроенный анализ трафика в реальном времени — раздел **Диагностика** в боковом меню:
- Все активные и закрытые соединения с цветовой меткой: VPN / Напрямую / Ошибка
- Фильтры: Все / Ошибки / Напрямую / VPN — с счётчиком в каждой вкладке
- Сортировка по времени, процессу, домену, ошибкам
- **Одним кликом** добавляй домен, процесс или IP в «Мои правила» прямо из списка соединений

### Собственные источники геоданных
В **Настройки → Геоданные и шаблоны** можно указать свои URL для `geosite.dat` и `geoip.dat`. По умолчанию используются базы [runetfreedom](https://github.com/runetfreedom/russia-v2ray-rules-dat) с актуальными списками заблокированных ресурсов РФ.

### Редактор шаблонов маршрутизации
Три режима маршрутизации основаны на YAML-шаблонах — их можно редактировать прямо в **Настройки → Геоданные и шаблоны**:
- Изменяй правила, группы прокси и любые параметры для каждого из трёх режимов
- Пользовательские шаблоны сохраняются в папке данных и применяются **горячей перезагрузкой** без перезапуска ядра
- Кнопка «Сбросить» возвращает шаблон к встроенному дефолту

### Мои правила — поддержка IP
В разделе **Мои правила** помимо доменов и процессов можно добавлять:
- 🔢 Одиночные IP-адреса (`1.2.3.4`)
- 🔢 Диапазоны CIDR (`192.168.0.0/16`, `2001:db8::/32`)

### Тёмная тема по умолчанию
Приложение запускается в тёмной теме без дополнительных настроек.

## Основные возможности

- **Три режима маршрутизации** — переключение в один клик:
  - 🔒 **Только заблокированные** — через VPN идут только сайты, заблокированные в РФ (Telegram, Discord и др.), остальное напрямую
  - 🌍 **Все зарубежные** — весь иностранный трафик через VPN, российские сайты напрямую
  - 🔁 **Всё через VPN** — полный туннель

- **Мои правила** — управление исключениями без редактирования конфигов:
  - Домены, приложения (по имени процесса), IP-адреса и CIDR-диапазоны
  - Импорт списком: вставь несколько записей сразу

- **Геоданные от [runetfreedom](https://github.com/runetfreedom/russia-v2ray-rules-dat)** — актуальные базы заблокированных ресурсов РФ, обновляются прямо в приложении и вшиты в установщик

## Установка подписки

1. Открой приложение → нажми **+** или перейди в **Профили**
2. Вставь ссылку на подписку → нажми **Сохранить**
3. Выбери режим маршрутизации и нажми кнопку подключения

## Расширенные настройки

### Режим Эксперта
**Настройки → Основные → Режим Эксперта** — скрывает технические разделы (Группы прокси, Подключения, Правила, Логи) для упрощённого вида. В режиме Эксперта открываются скрытые параметры: приоритет процесса, управление DNS, TUN-настройки и другие.

### Ручное редактирование конфига ядра
**Настройки → Ядро Mihomo** — редактор конфигурации ядра с подсветкой YAML и валидацией схемы.

### Геоданные и шаблоны маршрутизации
**Настройки → Геоданные и шаблоны**:
- Задай свои URL источников `geosite.dat` / `geoip.dat`
- Редактируй YAML-шаблоны для каждого режима маршрутизации
- Переименовывай режимы маршрутизации под свои нужды
